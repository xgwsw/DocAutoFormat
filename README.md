# 文档自动排版工具

基于 Java 的简易文档自动排版工具，支持文档读取、格式统一设置、冗余格式清理、文本统计等功能。

## 功能

- **文档读取**：支持 .txt / .md 格式，自动识别文件类型
- **格式统一设置**：字体（5种）、字号（4种）、行距（4种）、对齐（4种）
- **一键套用模板**：标准论文 / 公文格式 / 标题格式
- **冗余格式清理**：清除多余空行、行尾空格、连续空格
- **文本统计**：字数、段落数、行数、字符类型分布
- **友好的命令行菜单**：双层菜单，完善的错误处理

## 项目结构

```
DocAutoFormat/
├── src/
│   ├── Main.java          # 主程序入口
│   ├── Menu.java          # 菜单模块
│   ├── FileReader.java     # 文档读取模块
│   ├── FormatEngine.java   # 格式设置模块
│   ├── Cleaner.java        # 冗余清理模块
│   └── Document.java       # 文档数据模型
├── docs/                   # 设计文档
└── README.md
```

## 使用方法

```bash
# 编译
javac src/*.java

# 运行
java -cp src Main
```

按菜单提示选择功能：
1. 读取文档 → 2. 格式设置 → 3. 冗余清理 → 5. 保存文档
